# Installation
## Prerequisites
- nodejs
- npm
- mysql-server

## Dev-Preparation
- Clone git repository
```bash
git clone https://gitlab.ezlife.eu/ajo/ezcooking.git
```
- Initialize npm modules
```bash
npm install --save
```

- Start node server
```bash
node app.js
```
- Access Webapp
```bash
https://localhost:3000
```
